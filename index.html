Quick Start

                        
Write a pipeline starting from a source path, then chain steps.

                        
users | filter(@.age >= 21) | map({name, age})

                        
Common Steps

                        
filter(expr)
map(expr)
sort(path asc|desc)
limit(n), skip(n), at(n), slice(start,count)
distinct(path?), flatten(path?), count(expr?)

                        
Examples

                        
users | at(0)
users | skip(10) | limit(10)
users | distinct(profile.country)
orders | flatten(items) | map({id, sku: @.items.sku})

                        
Filtering Basics

                        
// exact match
users | filter(@.name == "Alice")

// numeric comparison
users | filter(@.age >= 21)

// null / exists checks
users | filter(@.profile != null)
users | filter(exists(@.profile.city))

// contains on strings
users | filter(contains(@.name, "li"))

// combine conditions
users | filter(@.age >= 21 && @.active == true)

                        
Tips

                        
@ = current item
$ = root JSON
For mixed logs/SSE text, JPL runs per detected JSON chunk.
